# Loading 上拉加载

一般用于上拉加载新数据场景使用

# list 数据结构

list: ['', '加载中...', '已加载全部信息']

# 全局配置（不配置则默认使用DNVUE内置）

开发者可全局自定义上拉加载数据，在 .dnvue/common/global.js 中配置，不配置则使用框架默认。

# 默认配置规范 按需配置

制作GIF动图 GO

module.exports = {
	//n-loading组件
	loading: {
		list: ['', '加载中...', '已加载全部信息'],
		loadingGif: 'xxxx.gif' //加载中GIF动图，不进行配置则使用DNVUE默认加载GIF动图
	}
}

# 基本使用

通过 status 参数来控制显示不同状态下的上拉加载

  • 0 : 空值（准备上拉加载）
  • 1 : 加载中...
  • 2 : 已加载全部信息
<template>
	<view>
		<n-loading status="1"></n-loading>
	</view>
</template>

# 自定义上拉加载数据

通过 list 参数自定义当前上拉加载数

<template>
	<view>
		<n-loading status="1" :list="['上拉加载更多', '疯狂加载中...', '没有更多信息了']"></n-loading>
	</view>
</template>

# API

# Attribute 参数

参数 说明 类型 可选值 默认值
status 上拉加载状态 String 0、1、2 0
list 上拉加载数据 String, Number -
loading-gif 加载中GIF图 String, Number - -
height 1.1.1 组件高度 String, Number - 88

# status 值

模式 说明
0 空值（准备上拉加载）
1 加载中...
2 已加载全部信息
