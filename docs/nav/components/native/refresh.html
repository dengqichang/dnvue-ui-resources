# refresh 下拉刷新

为容器提供下拉刷新功能。在nvue上，可通过此组件实现灵活的、自定义的、高性能的下拉刷新。

  • uni-app官方详细文档 (opens new window)
  • weex官方详细文档 (opens new window)

注意

<refresh> 是 <scroll-view>、<list>、<waterfall> 的子组件，只能在被它们包含时才能被正确渲染。

# 子组件

  • <loading-indicator>: 只能作为 <refresh> 和 <loading> 的子组件使用，拥有默认的动画效果实现。
<refresh>
  <text>Refreshing</text>
  <loading-indicator></loading-indicator>
</refresh>

# API

# Attribute 属性

属性 说明 类型 可选值 默认值
display 控制 <refresh> 组件显示、隐藏。display 的设置必须成对出现，即设置 display="show",必须有对应的 display="hide" string show / hide show

# Event 事件

事件名 说明 参数 返回值
@refresh 当 <scroll-view>、<list>、<waterfall> 被下拉完成时触发 - -
@pullingdown 当 <scroll-view>、<list>、<waterfall> 被下拉时触发。 - event
