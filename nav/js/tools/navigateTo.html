# navigateTo 页面跳转

此方法封装自uni的navigateTo跳转页面方法。调整页面动画兼容深色模式下的页面场景变换，简化携带参数。

# navigateTo(url,params)

url <String> 参数为跳转页面地址，params <Object> 参数为需要携带的数据

调用方式：

this.$store.tools.navigateTo(url,params)

跳转目标页面获取携带数据：

获取页面跳转方法携带的数据，方式：this.$store.state.params.跳转页面名称

# 使用示例

跳转页面

this.$store.tools.navigateTo("/pages/index/index",{name:"dnvue-ui"})

跳转目标页面

this.$store.state.params.index
//获取结果:{"name": "dnvue-ui"}
