# preloadPage 预加载

此方法封装自uni的preloadPage预加载，简化调整只兼容 H5 和 APP-NVUE 使用，可同时预加载多个页面。
uni.preloadPage(OBJECT) 官方文档 (opens new window)

# preloadPage(str)

str <String,Array> 参数可以是String类型和Arrar数组类型。String类型时只预加载一个页面，Array类型时同时预加载多个页面。

# String 类型方式

this.$store.tools.preloadPage('/pages/index/index');

# Arrar 类型方式

this.$store.tools.preloadPage(['/pages/index/index', '/pages/my/my']);
