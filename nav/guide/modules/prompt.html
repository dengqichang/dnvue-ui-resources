# HBuilder X代码提示

DNVUE UI 适配了HBuilder X代码提示，帮助开发者更快速、便捷的使用DNVUE组件，无需引入，使用DNVUE UI 即拥有。

# 使用方式

  1. 在页面代码中键盘敲 <n- 关键字，HBuilder X编译器会罗列出DNVUE的所有组件，通过选择组件的方式快速编写程序

  1. 选择好所需组件后，在组件中通过空格键将会弹出当前组件的参数及组件参数的描述，可快速选择参数调整组件

  1. 选择参数后，将会弹出当前参数可选值及参数值的描述
