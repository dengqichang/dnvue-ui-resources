# 文本粗细

class Properties 说明
text-normal font-weight: normal; 常规，normal 等同于 400
text-bold font-weight: bold; 加粗，bold 等同于 700

lines-2 text-overflow: ellipsis;lines: 2; 超出2行最大文本行数显示省略
lines-3 text-overflow: ellipsis;lines: 3; 超出3行最大文本行数显示省略
lines-4 text-overflow: ellipsis;lines: 3; 超出4行最大文本行数显示省略

# 文本对齐

class Properties 说明
text-left text-align: left; 把文本排列到左边
text-center text-align: center; 把文本排列到中间
text-right text-align: right; 把文本排列到右边

# 文本装饰

class Properties 说明
text-underline text-decoration: underline; 定义文本下的一条线。
text-through text-decoration: line-through; 定义穿过文本下的一条线。
