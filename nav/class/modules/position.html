# Position

class Properties 说明
absolute position: absolute; 绝对定位，以元素的容器作为参考系
absolute-screen position: absolute;left: 0;right: 0;top: 0;bottom: 0; 全屏，绝对定位，以元素的容器作为参考系
fixed position: fixed; 保证元素在页面窗口中的对应位置显示
fixed-screen position: fixed;left: 0;right: 0;top: 0;bottom: 0; 全屏，保证元素在页面窗口中的对应位置显示
sticky position: sticky; 指的是仅当元素滚动到页面之外时，元素会固定在页面窗口的顶部。
