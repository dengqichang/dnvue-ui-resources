# StatusBar 状态栏

一般用于在禁用原生导航栏后，需要来填充系统状态栏的高度

# 基本使用

通过 <n-status-bar> 形式调用，高度为系统状态栏高度

<n-status-bar></n-status-bar>

# 背景颜色

通过 background 参数来设置背景颜色，颜色参数并不局限内置颜色，也可以是颜色的本身的参数值，例如：#000000

<n-status-bar background="$theme-6"></n-status-bar>
