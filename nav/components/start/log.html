# 1.0.9（2021-04-13）

  • DNVUE 官方交流群：871507982
  • 【新增】<n-list> 列表组件，高性能长列表渲染组件，自定义下拉刷新、监听网络状态等特色功能
  • 【新增】<n-list-tab> 选项卡列表组件，用于选项卡切换场景下使用
  • 【新增】<n-block> 块组件，一般用于需设定宽、高情况或者渐变背景色等场景
  • 【新增】<n-line> 线条组件，一般用于显示一根线条，用于分隔内容块，有横向和竖向两种模式，多种线条模式（实线、虚线、点线）
  • 【新增】<n-divider> 区隔内容的分割线，一般用于页面底部"没有更多内容"的提示
  • 【修复】修复部分组件在小程序端报异常提示情况
  • 【优化】优化框架体积，从710KB降到450KB
  • 【优化】优化<n-empty> 为空组件可全局配置图片尺寸
  • 【优化】优化<n-swiper> 添加轮播图片未加载时的占位图
  • 【重点关注 * 优化】<n-page> 组件，添加监听网络状态功能，调整组件使用方式
  • 【重点关注 * 优化】<n-tabbar> 组件，调整组件使用方式
  • 【重点关注 * 优化】调整 DNVUE UI 框架配置方式，传送门
