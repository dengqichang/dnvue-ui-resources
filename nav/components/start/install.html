# 安装

说明

由于DNVUE使用uni_modules模式，让您无需引入组件即可直接使用，但是此功能需要Hbuilder X 3.1.0及以上版本才支持。
uni_modules打包的时候是按需引入的，您可以放心引入DNVUE的整个组件库，发布打包时会自动剔除您没有使用的组件

# 开发环境

请确保您下载的Hbuilder X (opens new window)为APP开发版，而非标准版

# 插件市场地址

插件市场地址：DNVUE UI (opens new window)

# 安装

在uni-app插件市场的DNVUE UI 多平台快速开发通用组件库 (opens new window)中选择 使用Hbuilder X导入插件 导入到项目中

# 配置

┌─根目录(项目名)
├───pages 
├───static 
├───uni_modules
├───App.vue        //第二步：引入index.css
├───main.js        //第一步：全局挂载框架配置
├───manifest.json 
├───pages.json  
└───uni.scss

# 第一步：main.js 配置

// 全局挂载
import dnvue from '@/uni_modules/dnvue-ui/index.js'
Vue.prototype.$store = dnvue

# 第二步：App.vue 中引入 index.css

<style>
	@import "@/uni_modules/dnvue-ui/index.css";
</style>

配置完成，使用愉快~^o^~

如果 DNVUE UI 能够帮助到您，请给个star (opens new window)和五星好评 (opens new window)支持鼓励一下作者吧，感谢支持与鼓励。

# 版本升级

在项目 uni_modules 目录下的 dnvue-ui 目录，鼠标右键选择 从插件市场更新 即可

升级前建议开发者先查看更新日志与升级指南的版本更新内容，以便开发者了解最新版本动态。
