# RichText 富文本解析 高性能渲染

一般用于富文本解析，例如新闻内容、文章内容、评论显示、推广信息等各类HTMl标签内容显示

# 平台差异说明

APP-NVUE APP-VUE H5 微信小程序 支付宝小程序 百度小程序 QQ小程序 字节跳动小程序
√ √ √ √ √ √ √ √

提示

当前 <d-rich-text> 组件为基础的渲染富文本数据（高性能渲染），如果需要更强大的富文本渲染组件，可查看d-html组件（渲染降级）

# 基本使用

通过 <d-rich-text> 形式调用，使用 content 参数进行渲染富文本

<d-rich-text :content="content" />

# API

# props 属性

属性 说明 类型 可选值 默认值
content 富文本内容 String - -
width 宽度(rpx) String,Number - -
min-width 最大宽度(rpx) String,Number - -

style-custom 组件自定义样式，同 css 样式。注意带’-‘连接符的属性需要使用小驼峰写法如：backgroundColor:red Object - -

注意

  • 支持默认事件，包括：click、touchstart、touchmove、touchcancel、touchend、longpress。
  • 全局支持class和style属性，不支持id属性。
