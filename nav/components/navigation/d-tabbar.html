# Tabbar 底部导航栏

此组件一般用于应用底部导航功能，可动态渲染自定义底部导航栏

# 平台差异说明

# list 参数解释

list: [{
	"iconPath": "d-home", //未选中图标（必填）
	"selectedIconPath": "d-homefill", //选中图标（必填）
	"text": "首页", //名称（必填）
	"count": '1', //数字角标（可选）
	"dot": false //红点（可选）
}]

# 基本使用

通过 list 数据动态渲染底部导航栏

<template>
	<view>
		<!-- 底部导航栏 -->
		<d-tabbar :list="tabbar" :current="current" @change="onTabbarChange"></d-tabbar>
	</view>
</template>

<script>
	export default {
		data() {
			return {
				current: 0,
				tabbar: [{
						"iconPath": "d-home",
						"selectedIconPath": "d-homefill",
						"text": "首页"
					},
					{
						"iconPath": "d-my",
						"selectedIconPath": "d-myfill",
						"text": "我的"
					}
				]
			}
		},
		onShow() {
			
		},
		methods: {
			onTabbarChange(e) {
				this.current = e.detail.current;
			}
		}
	}
</script>

# API

# props 属性

属性 说明 类型 可选值 默认值
list 底部导航栏数据源 Array, Object - -
current 控制当前显示的自定义导航栏页面 String, Number - 0
safetyArea 是否适配安全区域 Boolean false true
background 背景颜色 String, Object, Array $gray-1
borderTopColor 顶部边框颜色 String, Object, Array $gray-2
inactiveColor 未激活文本颜色 String, Object, Array $grey-6
activateColor 激活文本颜色 String, Object, Array $theme-6
textSize 文本大小 Number, String 22
iconSize 图标大小 Number, String 56

# Event 事件

事件名 说明 返回值
@change 底部导航栏切换时触发 当前切换导航栏下标
