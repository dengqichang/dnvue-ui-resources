# Tabs 选项卡

此组件为tabs选项卡，在选项卡多的时候，可以配置为左右滑动，选项卡少的时候，可以禁止滑动。 该组件的一个特点是配置为滚动模式时，激活的tab会自动移动到组件的中间位置，一般用于与选项页组件搭配使用。

# 基本使用

通过list参数配置，该参数要求为数组，元素为对象，对象要有name属性，见示例：

<template>
	<view>
		<d-tabs :current="current" :list="list" @change="onChange" />
	</view>
</template>

<script>
	export default {
		data() {
			return {
				list:[
					{name:'订阅'},
					{name:'精选'},
					{name:'发现'},
					{name:'电视剧'},
					{name:'电影'},
					{name:'综艺'},
					{name:'动漫'},
					{name:'少儿'},
					{name:'纪录片'}
				],
				current:0
			}
		},
		methods: {
			onChange(e){
				this.current = e.detail.current;
			}
		}
	}
</script>

# 控制组件读取的数组元素属性名

某些情况下，数据可能是从后端获取的，list 所需的数组中不一定会有 name 属性，比如可能为 tab_name，如果这种情况还需一定要提供 name 属性 会导致用户需要循环一遍， 把 tab_name 改成 name，显然不人性的，所以 DNVUE 给 d-tabs 组件提供了一个 name 参数，您可以设置其值为 tab_name ，组件内部会读取数组中的 tab_name 属性，而不是默认的 name 属性。

<template>
	<view>
		<d-tabs :current="current" name="tab_name" :list="list" @change="onChange" />
	</view>
</template>

<script>
	export default {
		data() {
			return {
				list:[
					{tab_name:'订阅'},
					{tab_name:'精选'},
					{tab_name:'发现'},
					{tab_name:'电视剧'},
					{tab_name:'电影'},
					{tab_name:'综艺'},
					{tab_name:'动漫'},
					{tab_name:'少儿'},
					{tab_name:'纪录片'}
				],
				current:0
			}
		},
		methods: {
			onChange(e){
				this.current = e.detail.current;
			}
		}
	}
</script>

# 控制底部滑块的样式

bar-width 控制滑块的长度(rpx)，bar-height 控制滑块高度(rpx)。

<d-tabs :current="current" :list="list" bar-width="30" bar-height="8" />

# 控制均分选项卡

通过设置 flex 属性为 true 即可均分选项卡

<d-tabs :current="current" :list="list" flex />

# API

# props 属性

属性 说明 类型 可选值 默认值
list 数据源 Array, Object - -
current 控制当前显示激活tabs选项 String, Number - 0
name 读取传入的数组对象的属性(tabs名称) String - name
size 字体大小 String, Number - 28
weight 激活字体加粗 Boolean false true
activeColor 激活项的颜色 String, Array, Object - $grey-10
inactiveColor 未激活项的颜色 String, Array, Object - $grey-10
flex 是否均分占位 Boolean true false
margin tabs 中 text的外边距，可用于控制每个选项卡之间的间距与高度，单位rpx String - 18 24 12
barHeight 底部bar的高度，单位rpx String, Number - 6
barWidth 底部bar的宽度，单位rpx String, Number - 40
barBackground 底部bar颜色 String, Array, Object - $theme-6

# Event 事件

事件名 说明 返回值
@change 切换时触发 当前切换下标
