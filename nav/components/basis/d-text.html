# Text 文本

用于显示文字，封装 text 文本组件，为兼容各端并快速实现文本的样式的处理。
推荐开发者小伙伴在任何使用文本显示场景的地方，都优先考虑使用这个小巧，精致，多功能实用的组件。

# 基本使用

通过 <d-text> 形式调用，设置 text 属性即可

<d-text text="明月几时有？把酒问青天。不知天上宫阙，今夕是何年。" />

# 超出行显示省略号

设置属性 lines 超出行数省略即可

<d-text lines="2" text="明月几时有？把酒问青天。不知天上宫阙，今夕是何年。我欲乘风归去，又恐琼楼玉宇，高处不胜寒。起舞弄清影，何似在人间。" />

# 文本位置

设置 align 属性来调整文本显示的位置

  • left : 左侧
  • center : 居中
  • right : 右侧
	<d-text align="center" text="明月几时有？把酒问青天。" />

# 文本装饰

设置 decoration 属性来装饰文本内容

  • none : 默认值，定义标准的文本。
  • line-through : 定义穿过文本下的一条线
  • underline : 定义文本下的一条线
<d-text decoration="line-through" text="明月几时有？把酒问青天。" />

# 文本行高

设置 leading 属性来调整文本行高， leading 属性为在原有的行高基础上加上行高，并非重新设置文本的行高。

<!-- 在原有的行高基础上加上 10rpx 行高 -->
<d-text leading="10" text="明月几时有？把酒问青天。" />

# 文本颜色

设置 color 属性为框架内置颜色或者自定义颜色 (opens new window)均可，框架内置颜色将自动适配暗黑模式下的颜色调整，推荐使用框架内置颜色

color 属性为 [String, Array] 类型：

  • String 类型：当 color 为String类型时，普通与暗黑模式下都是一样的颜色（自定义颜色）
  • Array 类型：当 color 为Array类型时，下标 0 为普通模式下显示的颜色，下标 1 为暗黑模式下显示的颜色（自定义颜色）
<d-text color="$theme-6" text="明月几时有？把酒问青天。不知天上宫阙，今夕是何年。" />
<!-- 或者 -->
<d-text color="#1890ff" text="明月几时有？把酒问青天。不知天上宫阙，今夕是何年。" />
<!-- 或者 -->
<d-text :color="['#1890ff','#ffffff']" text="明月几时有？把酒问青天。不知天上宫阙，今夕是何年。" />
<!-- 或者 -->
<d-text :color="['$theme-6','$red-6']" text="明月几时有？把酒问青天。不知天上宫阙，今夕是何年。" />

# 全局尺寸设置

通过设置 change 属性来分辨是否参与全局字体尺寸设置，默认值为 true ，参与全局尺寸调整。
如果当前显示内容不需要参与全局尺寸设置，则将 change 属性设置为 false 即可。
当前设置会影响 <d-icon> 组件，详情说明
通过调用api实现：

//设置全局参与尺寸调整的组件字体基础加大2rpx，参数需为Number数字类型。
this.$store.commit('dnvue/change', 2);

# Props 属性

属性 说明 类型 可选值 默认值
text 文本内容 String,Number - -
size 文本字号，单位:rpx String,Number - 28
color 文本颜色 String,Object,Array - $grey-10
lines 超出行省略 String, Number - -
align 文本位置 String left、center、right left
leading 行高，单位:rpx String,Number - -
weight 文本加粗 Boolean true false
decoration 文本修饰 String none、line-through、underline none
width 文本宽度，单位:rpx String,Number - -
margin 外边距，空格分割字符串形式简写，默认单位rpx。例如："12" 、 "12 24" String - -
change 是否参与动态尺寸设置 Boolean false true
style-custom 组件自定义样式，同 css 样式 Object - -
