# Image 图片

该组件封装加强 <image> 组件，除了原有的功能外，添加圆角，占位图等常用功能。
推荐开发者小伙伴在任何使用图片场景的地方，都优先考虑使用这个小巧，精致，多功能实用的组件。

# 平台差异

APP-NVUE APP-VUE H5 微信小程序 支付宝小程序 百度小程序 QQ小程序 字节跳动小程序
√ √ √ √ √ √ √ √

通过 <d-image> 形式调用，设置参数 src 显示图片

src 属性为 [String, Array] 类型：

  • String 类型：当 src 为String类型时，普通与暗黑模式下都是一样的图片
  • Array 类型：当 src 为Array类型时，下标 0 为普通模式下显示的图片，下标 1 为暗黑模式下显示的图片
<n-image src="图片地址" />
<!-- 或者 -->
<n-image :src="['普通模式图片地址','暗黑模式图片地址']" />

# 占位图

在图片下载过程中将展示占位图，图片下载完成后将显示src中指定的图片，通过 placeholder 参数进行配置占位图

placeholder 属性为 [String, Array] 类型：

  • String 类型：当 placeholder 为String类型时，普通与暗黑模式下都是一样的图片
  • Array 类型：当 placeholder 为Array类型时，下标 0 为普通模式下显示的图片，下标 1 为暗黑模式下显示的图片
<n-image placeholder="图片地址" />
<!-- 或者 -->
<n-image :placeholder="['普通模式占位图片地址','暗黑模式占位图片地址']" />

# 遮罩层

在暗黑模式下，图片需要深色处理，所以在暗黑模式下提供 mask 属性设置深色，透明底的图片不适用。

<d-image mask src="图片地址" />

# 全局尺寸设置

通过设置 change 属性来分辨是否参与全局字体尺寸设置，默认值为 false ，不参与全局尺寸调整。
通过设置 change-take 属性来调整参与全局字体大小设置的系数，例如 change-take 系数为1.5时，设置的全局加大为2，那当前使用的图片组件的加大尺寸即为1.5*2 = 3。
当前设置会影响 <d-text> 、<d-icon> 组件，详情说明
通过调用api实现：

//设置全局参与尺寸调整的组件字体基础加大2rpx，参数需为Number数字类型。
this.$store.commit('dnvue/textChange', 2);

# 图片裁剪、缩放的模式

mode 有 13 种模式，其中 4 种是缩放模式，9 种是裁剪模式。

模式 值 说明
缩放 scaleToFill 不保持纵横比缩放图片，使图片的宽高完全拉伸至填满 image 元素
缩放 aspectFit 保持纵横比缩放图片，使图片的长边能完全显示出来。也就是说，可以完整地将图片显示出来。
缩放 aspectFill 保持纵横比缩放图片，只保证图片的短边能完全显示出来。也就是说，图片通常只在水平或垂直方向是完整的，另一个方向将会发生截取。
缩放 widthFix 宽度不变，高度自动变化，保持原图宽高比不变
缩放 heightFix 高度不变，宽度自动变化，保持原图宽高比不变
裁剪 top 不缩放图片，只显示图片的顶部区域
裁剪 bottom 不缩放图片，只显示图片的底部区域
裁剪 center 不缩放图片，只显示图片的中间区域
裁剪 left 不缩放图片，只显示图片的左边区域
裁剪 right 不缩放图片，只显示图片的右边区域
裁剪 top left 不缩放图片，只显示图片的左上边区域
裁剪 top right 不缩放图片，只显示图片的右上边区域
裁剪 bottom left 不缩放图片，只显示图片的左下边区域
裁剪 bottom right 不缩放图片，只显示图片的右下边区域
