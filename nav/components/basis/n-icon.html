# Icon 图标

收集阿里字体图标，图标统一风格，满足大部分场景需求

提示

如果内置图标中不符合小伙伴的项目需求或者数量不够，别担心，我们只需在项目中进行简单的配置自定义的图标即可
我们还准备了一份简易的扩展自定义图标库教程

# 基本使用

通过 <d-icon> 形式调用，设置 name 参数为图标名称即可

<d-icon name="d-vip" />

# 自定义图标

按自定义图标库教程集成的自定义图标，也可以使用图标组件进行显示。
注意：因为dnvue-ui内置图标为 d- 前缀，所以自定义图标不可以 d- 为前缀，否则无效。

<d-icon name="自定义图标名" />

# 修改图标样式

  • color 参数修改图标颜色
  • size 参数修改图标的大小
<n-icon name="d-vip" color="$theme-6" size="42"></n-icon>

# 全局尺寸调整

通过设置 change-size 属性来分辨是否参与全局字体尺寸调整，默认值为 true ，参与全局尺寸调整。
如果当前显示内容不需要参与尺寸调整，则将 change-size 属性设置为 false 即可。
当前属性为通过调用api实现，一般用于调整全局字体尺寸时使用。

//设置全局参与尺寸调整的组件字体基础加大6rpx，参数需为Number数字类型。
this.$store.commit('dnvue/textChange', 6);

weight 加粗 Boolean true false
margin 外边距，空格分割字符串形式简写，默认单位rpx。例如："12" 、 "12 24"。 String - -
change-size 是否参与动态尺寸变化 Boolean false true
style-custom 自定义组件样式，同 css 样式，注意带’-‘连接符的属性需要使用小驼峰写法示例：:style-custom="{backgroundColor:red}" Object - -
