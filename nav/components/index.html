DNVUE UI

多平台快速开发 NVUE / VUE 组件库

# 简介

uni-app 2018年初发布至今，已经是业内最风靡的应用框架，支撑着800万开发者、数百万应用、12亿手机端月活用户、数千款 uni-app 插件的庞大生态。DNVUE UI 也是众多插件中的一员，目标是成为 uni-app 庞大生态圈中最优秀的框架。

关于 DNVUE 框架取名由来：首字母 D 来自于 DCloud 首字母，NVUE是基于 uni-app 中 NVUE 使用，故取名 DNVUE ，同时在此也对 DCloud 表示感谢。

# 适用领域

DNVUE UI是基于uni-app (opens new window)平台开发的（ NVUE / VUE ）UI框架，uni-app (opens new window)是一个使用 Vue.js 开发所有前端应用的框架， 开发者编写一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序（微信/支付宝/百度/头条/QQ/钉钉/淘宝）、快应用等多个平台。
