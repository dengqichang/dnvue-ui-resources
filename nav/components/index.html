DNVUE UI

多平台快速开发 NVUE / VUE 组件库

# 简介

DNVUE 致力于使开发者能基于uniapp (opens new window)跨平台框架快速开发APP / H5 / 各端小程序 / 快应用等多场景编译使用的UI框架。
在 nvue 页面模式采用 Weex 原生渲染引擎（提供了原生渲染能力），vue 页面模式采用 webview 渲染，DNVUE 并不强依赖任何特定页面模式，都同样兼容。 但是建议开发者使用 nvue 页面进行开发，nvue （Weex原生渲染能力）的性能远大于 vue （webview渲染）。

如果 DNVUE UI 能够帮助到您，请给个star (opens new window)和五星好评 (opens new window)支持鼓励一下作者吧，感谢支持与鼓励 ~^o^~

# 适用领域

DNVUE UI是基于uni-app (opens new window)平台开发的（ NVUE / VUE ）UI框架，uni-app (opens new window)是一个使用 Vue.js 开发所有前端应用的框架， 开发者编写一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序（微信/支付宝/百度/头条/QQ/钉钉/淘宝）、快应用等多个平台。
