# RichText 富文本解析 高性能渲染

一般用于富文本解析，例如新闻内容、文章内容、评论显示、推广信息等各类HTMl标签内容显示

提示

当前 <n-rich-text> 组件为基础的渲染富文本数据（高性能渲染），如果需要更强大的富文本渲染组件，可查看n-html组件（渲染降级）

# 基本使用

通过 <n-rich-text> 形式调用，使用 nodes 参数进行渲染富文本

<n-rich-text :nodes="nodes" />

# Attribute 参数

参数 说明 类型 可选值 默认值

  • 支持默认事件，包括：click、touchstart、touchmove、touchcancel、touchend、longpress。
