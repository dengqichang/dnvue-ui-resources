# Empty 为空展示

该组件用于需要加载内容，但是加载内容数据为空的情况下进行提示，精心挑选了十几种场景使用的缺省页状态

# 平台差异说明

APP-NVUE APP-VUE H5 微信小程序 支付宝小程序 百度小程序 QQ小程序 字节跳动小程序
√ √ √ √ √ √ √ √

# 全局自定义配置

开发者可全局自定义 为空展示 数据，在 .dnvue/common/empty.js 中进行配置

# 配置规范

module.exports = {
	mode参数: {
		src: "图片地址",
		describe: "描述内容",
		size:[宽度,高度]
	}
}

# 基本使用

通过 <n-empty> 形式调用，mode 参数设置显示为空数据展示类型，默认显示 没有内容

<n-empty mode="content"></n-empty>

# 自定义描述文本

通过 text 参数自定义当前缺省页的显示文本

<n-empty text="没有可显示的内容哦~"></n-empty>

# 嵌入内容

通过默认插槽可以在为空展示组件底部嵌入任意内容

<n-empty mode="shopping">
	<n-button width="300" text="去挑选"></n-button>
</n-empty>

# API

# props 属性

属性 说明 类型 可选值 默认值
mode 为空类型 String content / network / search / collection / evaluation / load / shopping / order / address / server content
text 描述文本 String, Number - -
size 文本字号 String, Number - 32
color 文本颜色 String - $gray-6

# mode 为空类型

mode 有 13 种模式，其中 4 种是缩放模式，9 种是裁剪模式。

模式 说明
content 内容缺省页
network 网络缺省页
search 搜索缺省页
collection 收藏缺省页
evaluation 评价缺省页
load 加载中缺省页
shopping 购物车缺省页
order 订单缺省页
address 地址缺省页
server 服务器缺省页
