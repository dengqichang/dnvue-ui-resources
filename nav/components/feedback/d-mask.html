# Mask 遮罩层

用于整个场景或一个特定区域，使场景外的对象或特定区域外的对象不可见，从而实现一些特殊的效果，一般用于在上级元素上层提示。

# 基本使用

通过 <d-mask> 形式调用，设置 show 属性控制显示或者关闭

<template>
	<view>
		<d-mask :show="true" />
	</view>
</template>

# 嵌入内容

通过默认插槽可以在遮罩层上嵌入任意内容，内容布局均为垂直居中。

<template>
	<view>
		<d-block width="500" height="500" background="#1890ff">
			<d-mask :show="true">
				<d-text text="我是遮罩层内部自定义内容" />
			</d-mask>
		</d-block>
	</view>
</template>

# API

# props 属性

属性 说明 类型 可选值 默认值
show 控制显示或者隐藏 Boolean true false
radius 圆角 String - -
background 遮罩层背景色 String,Array,Object - rgba(0, 0, 0, 0.4)

# Event 事件

事件名 说明 返回值
@change 当打开或者关闭时返回遮罩层状态 {detail: 遮罩层状态}
