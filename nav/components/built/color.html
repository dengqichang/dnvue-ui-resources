# Color 色彩

系统级色彩体系同样源于「自然」的设计价值观。设计师通过对自然场景的抽象捕捉，结合蚂蚁的技术基因，形成了特有的 12 色。进一步又通过大量的观察，捕捉不同色彩在自然光下的变化规律，借助美术中素描的思路，对 12 个颜色进行了衍生。在中性色板的定义上，则是平衡了可读性、美感以及可用性得出的。

# 基础色板

基础色板共计 120 个颜色，包含 12 个主色以及衍生色。这些颜色基本可以满足中后台设计中对于颜色的需求。

色彩说明

  • 所有组件中只要是涉及到使用颜色的属性，均可使用内置色彩（例如： $theme-6），也可使用自定义BGR颜色值（例如：#1890ff）。
  • 内置色彩均会参与暗黑模式适配方案，自定义GBR颜色值不参与暗黑模式适配方案。

# 组件示例

<d-text color="$theme-6" text="内置颜色" />
<!-- 或者 -->
<d-text color="#1890ff" text="RGB颜色" />

# 调整内置颜色

以下色彩均为默认内置颜色，如需调整可在下方颜色选择区域调整自己所需颜色，然后复制设置颜色方法放置App.vue页面中onLaunch函数中即可生效。

//设置主题色
this.$store.commit('dnvue/color',{
	theme:'#1890ff'
	//...支持同时修改多种内置颜色，例如以下方式：
	//theme:'#1890ff',red:'#f5222d'
});
