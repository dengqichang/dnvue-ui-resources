# 介绍

此专题的js函数方法，为DNVUE框架提供核心功能，是框架不可或缺的部分。

注意：

  1. 支持 nvue / vue 页面全局注册挂载（也就是说uni-app官方原本APP-NVUE页面是不能全局挂载js的，需在nvue当前页面引入才可调用外部js函数，但是在DNVUE框架中，已经自动处理并可以全局注册挂载，但需按照DNVUE框架的规范去实现）
  2. 在全端（APP-NVUE、APP-VUE、H5、各端小程序）支持全局注册挂载基础上，规范全端的函数方法统一调用形式，为框架全端兼容（nvue / vue）奠定了统一的基础。
  3. JS工具库包罗万象，引入uni-app插件市场中一些优秀的开源作品，为开发者们打开便利的通道。

# JS

如果是在js中，需要通过 this.$store.xxx 形式调用，如调用查看当前版本的 version() 方法：

//查看当前版本
console.log(this.$store.tools.version())

# template

如果是在元素中，无需前缀this，如：

<template>
	<view>
		<!-- 获取当前版本号 -->
		{{$store.tools.version()}}
	</view>
</template>
