# Badge 数字角标

一般用于显示未读的消息数量，提示用户点击。

# 基本使用

通过 <n-badge> 形式调用，设置 count 参数即可

<n-badge count="6"></n-badge>

# 封顶的数字值

通过设置 caps-count 参数封顶值，当 count 值超过封顶值后，将会显示 封顶值+

<n-badge count="10" caps-count="9"></n-badge>

# 显示为红点

通过设置 is-dot 参数为 true 即可

<n-badge is-dot></n-badge>

# 修改样式

通过 color 设置数字文本颜色，background 设置数字角标背景颜色。颜色值可内置颜色，也可HexColor值

<n-badge color="#FFFFFF" background="#f5222d"></n-badge>
<!-- 或者 -->
<n-badge color="$gray-1" background="$red-6"></n-badge>

# API

# Attribute 参数

参数 说明 类型 可选值 默认值
count 展示的数字 Number, String - 0
caps-count 展示封顶的数字值 Number, String true 99
size 角标尺寸 Number, String - 35
is-dot 显示为红点 Boolean true false
is-dot-size 红点尺寸 Number, String - 15
color 文本颜色 String - $gray-1
background 背景颜色 String - $red-6
