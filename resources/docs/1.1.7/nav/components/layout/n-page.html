# Page 页面

更高效便捷的完成页面布局划分，划分 navbar (自定义导航栏区域)、header (头部区域)、body (主体区域) 、footer (底部区域)、other (放置弹出层、mask、全屏弹出等场景下使用)。

# Slot 插槽

name 说明
navbar 自定义导航栏区域
header 头部区域
body 主体区域
footer 底部区域
other 放置弹出层、mask、全屏弹出等场景下使用

# 基本使用

通过 <n-page> 形式调用，使用各层级的 <slot> 划分的区域

注意

在 slot 为 body 时需使用 <n-list> 组件，传送门

<template>
	<n-page>
		<!-- 自定义导航栏区域 -->
		<view slot="navbar"></view>
		<!-- 头部区域 -->
		<view slot="header"></view>
		<!-- 主体区域 -->
		<n-list slot="body">
			<n-cell></n-cell>
		</n-list>
		<!-- 底部区域 -->
		<view slot="footer"></view>
	</n-page>
</template>
