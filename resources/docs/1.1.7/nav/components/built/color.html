# Color 色彩

系统级色彩体系同样源于「自然」的设计价值观。设计师通过对自然场景的抽象捕捉，结合蚂蚁的技术基因，形成了特有的 12 色。进一步又通过大量的观察，捕捉不同色彩在自然光下的变化规律，借助美术中素描的思路，对 12 个颜色进行了衍生。在中性色板的定义上，则是平衡了可读性、美感以及可用性得出的。

# 基础色板

基础色板共计 120 个颜色，包含 12 个主色以及衍生色。这些颜色基本可以满足中后台设计中对于颜色的需求。

# 色彩说明

在DNVUE框架中，所有组件中只要是涉及到使用颜色的属性，均可使用内置色彩（例如： $theme-6），也可使用BGR颜色值（例如：#1890ff）

<n-text color="$theme-6" text="内置颜色" />
<!-- 或者 -->
<n-text color="#1890ff" text="RGB颜色" />
