# 1.1.7（2021-07-01）

# 【补充版本】@1.1.7为补充版本，因新版本的dnvue-ui框架资源目录调整，所以@1.1.5版本之前的dnvue-ui需更新到@1.1.7补充版本。主要是调整框架资源文件，比如为空组件、扩展富文本组件资源，确保资源文件的正常使用。

# 【重要说明】@1.1.7版本为【旧版本】，之后更新的【新版本】将不向下兼容，所以使用@1.1.7版本及之前版本请勿更新@1.1.8新版本及往后版本。

# 系统级色彩体系

系统级色彩体系同样源于「自然」的设计价值观。设计师通过对自然场景的抽象捕捉，结合蚂蚁的技术基因，形成了特有的 12 色。进一步又通过大量的观察，捕捉不同色彩在自然光下的变化规律，借助美术中素描的思路，对 12 个颜色进行了衍生。在中性色板的定义上，则是平衡了可读性、美感以及可用性得出的。
