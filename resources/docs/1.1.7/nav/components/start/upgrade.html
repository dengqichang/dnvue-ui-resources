# 升级指南

升级指南中只会写入当前升级版本需要留意与特殊处理变更的地方，详细的版本升级日志请移步更新日志

# 1.1.1 升级指南（2020-04-19）

  • 【重点关注 * 优化】使用vuex的开发者注意，需重新调整vuex状态管理的使用方式，可一键搜索并替换原有项目中的 $store.vuex 替换成 $store即可，调整之后的vuex用法与vuex官方文档一致。

# 1.0.9 升级指南（2020-04-13）

  • 【重点关注】调整 DNVUE UI 框架配置方式，传送门
  • 【重点关注】调整 <n-page> 组件使用方式，升级1.0.9版本后原1.0.8版本之前<n-page>组件将不可用，需按新1.0.9版本后<n-page>组件进行调整。
  • 【重点关注】调整 <n-tabbar>组件使用方式，升级1.0.9版本后原1.0.8版本之前<n-tabbar>组件将不可用，需按新1.0.9版本后<n-tabbar>组件进行调整。
