# 安装

说明

由于DNVUE使用uni_modules模式，让您无需引入组件即可直接使用，但是此功能需要Hbuilder X 3.1.0及以上版本才支持。
uni_modules打包的时候是按需引入的，您可以放心引入DNVUE的整个组件库，发布打包时会自动剔除您没有使用的组件

# 开发环境

请确保您下载的Hbuilder X (opens new window)为APP开发版，而非标准版， 并且在"工具-插件安装"中安装了"scss/sass编译 (opens new window)"插件

# 下载地址

下载地址：DNVUE UI v.1.1.7 (opens new window)

# 配置

# 1. main.js 中配置

// 全局挂载
import config from '@/uni_modules/dnvue-ui/config.js'
Vue.prototype.$store = config

# 2. uni.scss 中引入 theme.scss

@import "@/uni_modules/dnvue-ui/css/theme.scss";

# 3. App.vue 文件中引入 index.scss，注意: 需要在style上添加lang="scss"标签

<style lang="scss">
	@import "@/uni_modules/dnvue-ui/css/index.scss";
</style>

# 4. 将 .dnvue 文件目录放置根目录(项目名)下（与App.vue文件同级）

  • 方式一：在 根目录(项目名)/uni_modules/dnvue-ui/ 路径下有一份 .dnvue 文件目录，复制到根目录(项目名)下即可。
┌─根目录(项目名)
├───.dnvue          //第四步：放入.dnvue目录文件
|     ├─common
|     └─vuex
├───pages 
├───static 
├───uni_modules
├───App.vue        //第三步：引入index.scss
├───main.js        //第一步：全局挂载框架配置
├───manifest.json 
├───pages.json  
└───uni.scss       //第二步：引入theme.scss

# 配置完成，使用愉快~^o^~

# 版本升级

在项目 uni_modules 目录下的 dnvue-ui 目录，鼠标右键选择 从插件市场更新 即可

升级前建议开发者先查看更新日志与升级指南的版本更新内容，以便开发者了解最新版本动态。
